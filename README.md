# Summa

## A simple spreadsheet for Early Modern British currency

[**Use Summa**](https://gjhilton.github.io/Summa/)

Summa is a simple spreadsheet for historians working with Early Modern British ledgers, accounts and similar documents.

<img src="src/assets/problem.png" width="100%" alt="Early modern ledger manuscript" />

<sup>Source: <a href="https://digitalcollections.folger.edu/img59191#main-content">Folger Shakespeare Library Digital Collections</a></sup>

For clerks of the era, summing columns of figures expressed as pounds, shillings and pence in Roman numerals was second nature. For modern users the calculations can be error-prone, and in large quantities quickly become tedious. Summa automates the calculation (and allows you to follow the working out, if you like).
